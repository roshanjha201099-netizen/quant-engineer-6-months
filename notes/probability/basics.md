# Probability Basics

Probability = favorable outcomes / total outcomes

Expected value:
E[X] = sum(value * probability)

Example:
Fair die roll:
E[X] = (1+2+3+4+5+6)/6 = 3.5

Variance:
Var(X) = E[X²] - E[X]²
